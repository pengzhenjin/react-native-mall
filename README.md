### 运行项目

- iOS，在项目根目录下依次执行如下命令：
```
yarn install
```

```
cd ios
pod install
```

```
yarn ios
```

- Android，在项目根目录下依次执行如下命令：
```
yarn install
```

```
yarn android
```
